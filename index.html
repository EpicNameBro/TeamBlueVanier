SETUP

  1. go to https://windows.github.com to download git hub
  2. create a folder in your "My documents" called TeamBlueVanier
  3. After installing git, run it, login and click on the arrow on the top left to add a repository
  4. In this menu, chose the option clone and select TeamBlueVanier
  5. You will be prompted to select a folder to clone to. Select the TeamBlueVanier folder you just created

HOW TO COMMIT

  1. modify or add a file in the TeamBlueVanier folder.
  2. go in git and click on the arrow next to "Uncommitted changes"
  3. Enter a summary and description of the changes you made and click "commit to.."
  4. Click on the sync button on the top left
